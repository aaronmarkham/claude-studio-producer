# Podcast Segment Types

Reference document for podcast segment structures learned from training data.

## Core Segments

### Intro (10-20s)
- Brand/show identification
- Host introduction
- Sets tone and energy level
- Visual: Logo animation, host establishing shot

### Hook (20-45s)
- Compelling question or statement
- Preview of key insight
- Creates curiosity gap
- Visual: Dynamic cuts, text overlays

### Main Content (120-300s)
- Core information delivery
- Structured into digestible chunks
- Includes examples and analogies
- Visual: Explainer graphics, demonstrations

### Callback (15-30s)
- References hook promise
- Delivers the payoff
- Creates satisfaction
- Visual: Callback to opening imagery

### Outro (10-20s)
- Summary of key points
- Call to action
- Closing brand elements
- Visual: End card, social handles

## Segment Transitions

### Hard Cut
- Used between major topic shifts
- Creates clear separation

### Crossfade
- Used within topics
- Smooths time jumps

### Motion Wipe
- Used for energy moments
- Adds dynamism

## Segment Validation Rules

1. Every podcast must have intro and outro
2. Main content must be longest segment
3. Hook should appear within first 45 seconds
4. Total duration should match target within 10%
